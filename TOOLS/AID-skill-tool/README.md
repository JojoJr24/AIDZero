# AID Skill Tool

CLI tool to interact with Agent Skills under `SKILLS/`.

## Commands

```bash
uv run python TOOLS/AID-skill-tool/tool.py list
uv run python TOOLS/AID-skill-tool/tool.py show --skill AID-skill-creator
uv run python TOOLS/AID-skill-tool/tool.py run-script --skill AID-skill-creator --script quick_validate.py -- SKILLS/AID-skill-creator
```
